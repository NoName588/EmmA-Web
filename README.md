# EmmA-Web

Coloca tus imágenes de perfil y proyectos en la carpeta `assets/`. Por ejemplo:

- `assets/profile.png` → Foto o avatar circular para el header.
- `assets/project1.png`, `assets/project2.png` → Imágenes representativas de tus proyectos para la grilla de portafolio.

Puedes cambiar estos nombres y rutas según tus necesidades.
